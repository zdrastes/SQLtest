# SQL Database Explorer

Интерактивное веб-приложение для изучения SQL и баз данных с практическими заданиями.

## 📋 Описание проекта

SQL Database Explorer - это образовательная платформа, которая предоставляет:
- 4 варианта практических заданий по SQL
- Интерактивное отображение структуры баз данных
- Демонстрацию данных таблиц
- Поля для ввода SQL-запросов
- Систему авторизации для доступа к ответам

## 🚀 Возможности

### Для студентов
- **4 варианта заданий**: товары и склады, сотрудники и проекты, университет, интернет-магазин
- **Интерактивные схемы баз данных**: наглядное отображение таблиц и связей
- **Просмотр данных**: возможность изучать содержимое таблиц
- **Практика**: поля для написания собственных SQL-запросов
- **Ответы**: доступ к правильным решениям после авторизации

### Для преподавателей
- **Система авторизации**: контроль доступа к ответам
- **Разные варианты**: возможность давать разные задания студентам
- **Удобная проверка**: структурированные ответы для каждого задания

## 🔐 Настройка пользователей

Для работы системы авторизации необходимо создать файл `users.json` в корневой директории проекта. Используйте файл `users.example.json` как шаблон:

```bash
# Создайте файл users.json на основе примера
copy users.example.json users.json
```

Затем отредактируйте `users.json` и добавьте нужных пользователей:

```json
[
    {
        "id": 1,
        "username": "your_username",
        "password": "your_password",
        "email": "user@example.com",
        "fullName": "Ваше Имя",
        "role": "student",
        "createdAt": "2025-01-01",
        "description": "Описание"
    }
]
```

**⚠️ Важно:** В учебных целях пароли хранятся в открытом виде. В реальном проекте необходимо использовать хеширование паролей!

## 📦 Установка и запуск

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   cd SQLtest-main
   ```

2. **Запуск проекта**
   - Просто откройте `index.html` в браузере
- Или используйте локальный сервер:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (с http-server)
     npx http-server
     ```

3. **Использование**
   - Выберите вариант задания на главной странице
- Изучите схему базы данных
- Просмотрите данные таблиц
- Напишите SQL-запросы в соответствующие поля
- Авторизуйтесь для просмотра правильных ответов

## 📝 Примеры SQL-запросов

### Простой SELECT
```sql
SELECT name, brand
FROM goods
WHERE brand = 'Xiaomi';
```

### JOIN запрос
```sql
SELECT w.name, g.name, s.quantity
FROM warehouses w
JOIN stock s ON w.id = s.warehouse_id
JOIN goods g ON s.goods_id = g.id
WHERE w.name = 'Варшавка';
```

### Агрегатные функции
```sql
SELECT brand, COUNT(*) as количество
FROM goods
GROUP BY brand;
```

Вариант 1 - База данных товаров и складов

Схема базы данных

goods

Таблица товаров

id INT PRIMARY KEY
name VARCHAR(100)
type_id INT
brand VARCHAR(100)

stock

Таблица остатков на складах

goods_id INT
warehouse_id INT
quantity INT

warehouses

Таблица складов

id INT PRIMARY KEY
name VARCHAR(100)
address VARCHAR(100)

types

Таблица типов товаров

id INT PRIMARY KEY
name VARCHAR(100)

Таблицы связаны между собой через ключи: goods.id → stock.goods_id, warehouses.id → stock.warehouse_id, types.id → goods.type_id

SQL Запросы к базе данных

1 Вывести количество артикулов (уникальных единиц товара) бренда Xiaomi. Столбцы «бренд», «количество».
2 Вывести все товары брендов Xiaomi и Adidas. Вывести колонки имя_товара, бренд_товара.
3 Заменить имя товара «Наушники беспроводные Xiaomi Airdots 2» на «Наушники беспроводные Huawei Airdots» и бренд на «Huawei».
4 Очистить полностью таблицу «warehouses».
5 Вывести все товары, которые лежат на складах: Варшавка, Пушкино. Вывести колонки имя_склада, имя_товара, кол-во_товара.
6 Наполнить таблицу «warehouses» данными, используя транзакцию.
7 Вывести количество товаров на всех складах с типами «Обувь» и «Аксессуары для телефонов». Столбцы «тип товара» и «количество».
